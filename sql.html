SQL

Resumo:


Pré-requisitos

Compreender os conceitos da Lógica de Programação

Neste curso você irá aprender a:

  • Entender o propósito e funcionamento de um Banco de Dados Relacional.

  • Estruturar corretamente as diferentes possibilidades de SQL.

  • Criar, excluir, e entender quais os tipos de dados de um banco de dados.

  • Modelar um banco de dados a partir da análise de negócios.

  • Manipular a informação contida nas entidades.

  • Criar consultas das mais simples até consultas com filtros complexos e diversas tabelas.

  • Utilizar as principais funções predefinidas dos bancos de dados Oracle e Microsoft SQL Server.

  • Realizar BackUp e Restore simples dos bancos de dados Oracle e Microsoft SQL Server.

  • Compreender as possibilidades de carreira que o mercado oferece ao profissional de Banco de Dados.

Público-Alvo: Programadores ou estudantes de programação e entendedores de lógica de negócios que busquem conhecimento para criar, manter e extrair informações de um banco de dados, seja SQL Server ou Oracle.

Críterio de aprovação

Média = MB1 + MB2 2 em que: MB1 (nota do primeiro bimestre) = 0,7 * P1 + 0,3 * ME1

P1 = nota da primeira avaliação

ME1 = nota de trabalho do primeiro bimestre



MB2 (nota do segundo bimestre) = 0,7 * P2 + 0,3 * ME2

P2 = nota da segunda avaliação

ME2 = nota de trabalho do segundo bimestre



MF (média final) = Média aritmética entre MB1 e MB2


MS=(MB1 +MB2) / 2

SE MS >=7,0 e Freqüência>=75% das aulas dadas ENTÃO

Aluno está aprovado

SENÃO SE MS>=3,0 e Freqüência >=75% ENTÃO

Aluno pode fazer o exame final

MF Exame = (MS + NotaExame)/2

SE MF Exame >=5,0 ENTÃO

Aluno está aprovado

SENÃO Aluno está Reprovado SENÃO Aluno está Reprovado, sem direito ao exame final.
