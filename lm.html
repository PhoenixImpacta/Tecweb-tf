LIMGUAGEM DE PROGRAMAÇÃO

Resumo:


A programação consiste na criação de um conjunto de instruções, um programa, de modo a possibilitar que o computador execute uma determinada tarefa ou atinja algum objetivo específico. O programa deve, portanto, ser escrito em uma linguagem de programação. Existem diversas linguagens de programação, que funcionam de modos diferentes e que podem criar diversos e diferentes programas.

SOBRE:

Criar softwares e desenvolver comandos para os computadores executarem certas tarefas são as principais funções do programador. Para trabalhar, este profissional precisa ter conhecimento de diferentes linguagens como Java, PHP entre outros. Boa parte dos profissionais que atuam na área de programação são autodidatas. No entanto, o acesso ao conhecimento é mais facilitado. A dica para quem pretende atuar nesse ramo promissor é começar realizando cursos gratuitos online de programação. Dessa forma, é possível ter acesso a novos conhecimentos e saber se é realmente essa carreira que você deseja, sem precisar pagar nada. Além disso, ainda vale como atividade complementar se você for universitário.

Pré-requisitos

Ensino Médio completo

Críterio de aprovação

Média = MB1 + MB2 2 em que: MB1 (nota do primeiro bimestre) = 0,7 * P1 + 0,3 * ME1

P1 = nota da primeira avaliação

ME1 = nota de trabalho do primeiro bimestre



MB2 (nota do segundo bimestre) = 0,7 * P2 + 0,3 * ME2

P2 = nota da segunda avaliação

ME2 = nota de trabalho do segundo bimestre



MF (média final) = Média aritmética entre MB1 e MB2


MS=(MB1 +MB2) / 2

SE MS >=7,0 e Freqüência>=75% das aulas dadas ENTÃO

Aluno está aprovado

SENÃO SE MS>=3,0 e Freqüência >=75% ENTÃO

Aluno pode fazer o exame final

MF Exame = (MS + NotaExame)/2

SE MF Exame >=5,0 ENTÃO

Aluno está aprovado
