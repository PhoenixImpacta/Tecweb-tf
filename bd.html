BANCO DE DADOS

Resumo:


O curso online de SQL 2016 - Criando Sistemas de Banco de Dados é ideal para desenvolvedores que pretendem criar e gerenciar um sistema de banco de dados em SQL Server. Durante o curso, você aprenderá a criar um banco de dados e utilizar diversos recursos do SQL Server e sua estrutura de consulta e manipulação de dados.

SOBRE:

SQL 2016 - Criando Sistemas de Banco de Dados (online) é ministrado por Hélio de Almeida, consultor em soluções de desenvolvimento de aplicações .NET, VBA e banco de dados SQL Server, e soluções em análises de estratégia de negócios (BI). Formado em gestão da tecnologia da informação, é instrutor desde 1994 e ministra cursos de VBA, C# e SQL.

Pré-requisitos

Para que você possa obter um bom aproveitamento do curso de SQL 2016 - Criando Sistemas de Banco de Dados (online), é imprescindível ter familiaridade com o ambiente Windows e conhecimentos de lógica de programação. Também, é importante possuir o SQL Server 2016 e o SSMS (SQL Server Management Studio) 2016 instalados em seu computador.

Críterio de aprovação

Média = MB1 + MB2 2 em que: MB1 (nota do primeiro bimestre) = 0,7 * P1 + 0,3 * ME1

P1 = nota da primeira avaliação

ME1 = nota de trabalho do primeiro bimestre



MB2 (nota do segundo bimestre) = 0,7 * P2 + 0,3 * ME2

P2 = nota da segunda avaliação

ME2 = nota de trabalho do segundo bimestre



MF (média final) = Média aritmética entre MB1 e MB2


MS=(MB1 +MB2) / 2

SE MS >=7,0 e Freqüência>=75% das aulas dadas ENTÃO

Aluno está aprovado

SENÃO SE MS>=3,0 e Freqüência >=75% ENTÃO

Aluno pode fazer o exame final

MF Exame = (MS + NotaExame)/2

SE MF Exame >=5,0 ENTÃO

Aluno está aprovado
